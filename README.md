# 🚀 AI-Powered Sales Analytics Pipeline

> Fully automated sales reporting pipeline built with n8n, Python and Google Gemini AI

## 📌 What It Does
- ⏰ Runs automatically on a schedule
- 📊 Fetches 70+ sales records from Google Sheets
- 🐍 Python calculates all KPIs instantly
- 🤖 Gemini AI generates beautiful HTML charts
- 📧 Sends daily email report to inbox
- 📋 Auto-updates Dashboard sheet

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| n8n | Workflow Automation |
| Python | Data Processing | claude sonnet 4.6 |
| Google Gemini AI | Chart Generation |
| Google Sheets | Data Source + Dashboard |
| Gmail | Automated Reports |

## 🔄 Pipeline Flow
Schedule Trigger
→ Google Sheets (fetch data)
→ Python (process & calculate)
→ Gemini AI (generate charts)
→ Google Sheets (update dashboard)
→ Gmail (send report)
